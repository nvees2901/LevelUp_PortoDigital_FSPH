# FSPH---Squad-02

##  Sobre o Projeto

Este projeto tem como objetivo desenvolver um sistema inteligente para **análise e validação de Termos de Referência**, com base na **Lei nº 14.133 (Nova Lei de Licitações e Contratos Administrativos)**.

A plataforma utilizará **Inteligência Artificial** para auxiliar na elaboração, validação e análise técnica dos termos, reduzindo o tempo do processo administrativo e aumentando a conformidade com a legislação.

O sistema também permitirá a **geração de novos termos de referência a partir de uma base existente**, por meio de um **chat inteligente**.

---

##  Objetivos

- Reduzir o tempo de elaboração e análise de termos de referência
- Auxiliar na validação automática com base na Lei 14.133
- Apoiar a análise técnica dos documentos
- Permitir a geração automática de novos termos utilizando IA
- Criar uma base estruturada de termos de referência para consulta

---

##  Funcionalidades

###  Chat Inteligente
Interface onde o usuário poderá conversar com a IA para:
- Gerar novos termos de referência
- Consultar termos existentes na base
- Receber sugestões e melhorias no documento

###  Análise Automática de Termos
O sistema será capaz de:
- Validar o termo com base na Lei nº 14.133
- Realizar análise técnica do conteúdo
- Identificar possíveis inconsistências ou melhorias

###  Cadastro de Novos Termos
Interface para:
- Adicionar novos termos de referência à base
- Atualizar e manter o banco de termos utilizado pela IA
